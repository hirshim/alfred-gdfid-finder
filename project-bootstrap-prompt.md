## プロジェクト概要

- 名前: alfred-gdfid-finder
- 種類: スクリプト
- 目的: 
  - 選択した文字列をGoogle ドライブのファイルIDとして解釈し、Finder上で該当するファイルを選択された状態にする。
  - Obsidian上にあるGoogleドライブのファイルデータベスから、該当するファイルに簡単にアクセスしたい

## 技術スタック

- Alfred Workflow
- Python 3.9.6
- パッケージ管理: uv
- テスト: pytest
- リンター: ruff
- 型チェック: mypy

## コーディング方針

- docstring: Google style
- 型ヒント必須
- テストカバレッジ目標: 80% 以上

## 開発プロセス（CLAUDE.md に含める）

### 基本ルール

- レビュー（変更無し、Ultrathink）→ リファクタ（リンター、型チェック、ユーザーに質問） → レビュー のサイクルを、変更点がなくなるまで続ける
- 開発ステップごとに、プロジェクト全体のレビューをおこなう

### ドキュメント管理

- ドキュメントは日本語で記述する
- CLAUDE.md ファイルには「関連ドキュメント」セクションを設ける
	- 各ファイルのパスと役割を明記する
- CLAUDE.md ファイルは 200 行以内を保つ
	- 200 行を超える場合は、ユーザーに分割方針を質問する
	- 分割する場合：
		- ファイル名は `docs/CLAUDE-**.md` 形式とする
